# Out of Scope

The following are explicitly excluded from Phase 1:

- **Production execution**: Running generated SAS/R programs against real study data.
- **Multi-study orchestration**: Managing multiple studies simultaneously.
- **Web UI / dashboard**: No browser-based interface in Phase 1.
- **Visual mind-map / graph view**: Deferred to Phase 2+; the structured data model supports future visualization.
- **Real-time collaboration**: No multi-user concurrent editing.
- **Cloud deployment**: Phase 1 is local-only.
- **CDISC / SDTM / ADaM submission packaging**: Out of scope for the knowledge base phase.
- **Patient data handling**: No real patient data should enter this system.
- **Automated promotion**: Assets cannot skip human review to reach production-approved status.
