# Document Types

Types of documents that may be referenced or stored in the Clinical Coding Center.

## Primary Reference Documents

| Type | Description | Typical Use |
|------|-------------|-------------|
| SAP | Statistical Analysis Plan | Defines analyses, populations, endpoints |
| Mock Shell | Empty table/listing template with headers and structure | Defines expected output format |
| Annotated CRF | CRF pages with SDTM variable annotations | Maps collected data to standard variables |
| Data Specification | Dataset and variable definitions | Describes available data structures |
| Programming Specification | Detailed instructions for a specific program | Step-by-step derivation rules |

## Supporting Documents

| Type | Description |
|------|-------------|
| Protocol | Study protocol (design, objectives, endpoints) |
| Protocol Amendment | Changes to the original protocol |
| Data Review Plan | How data quality is monitored |
| Validation Plan | How programs are validated (double programming, QC) |

## Internal Documents

| Type | Description |
|------|-------------|
| Review Guide | Instructions for reviewing generated output |
| Coding Standard | Programming conventions and style rules |
| Runbook | Step-by-step operational procedures |
| Troubleshooting Guide | Common problems and solutions |
