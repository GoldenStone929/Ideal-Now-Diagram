# Terminology

Common terms used across the Clinical Coding Center.

## Clinical Terms

| Term | Abbreviation | Definition |
|------|-------------|------------|
| Adverse Event | AE | Any untoward medical occurrence in a subject |
| Serious Adverse Event | SAE | AE that results in death, hospitalization, etc. |
| Treatment-Emergent AE | TEAE | AE with onset after first dose of study treatment |
| System Organ Class | SOC | MedDRA high-level grouping of AEs |
| Preferred Term | PT | MedDRA standardized AE term |
| Intent-to-Treat | ITT | All randomized subjects |
| Safety Population | SAF | All subjects who received at least one dose |
| Per-Protocol | PP | Subjects who completed the study without major protocol deviations |
| Statistical Analysis Plan | SAP | Document describing planned analyses |
| Case Report Form | CRF / eCRF | Data collection instrument for clinical trials |
| Big N | N | Column header denominator (total subjects in a group) |
| Small n | n | Count of subjects meeting a specific criterion |

## Data Standards Terms

| Term | Definition |
|------|------------|
| SDTM | Study Data Tabulation Model (raw data standard) |
| ADaM | Analysis Data Model (analysis-ready data standard) |
| CDISC | Clinical Data Interchange Standards Consortium |
| ADSL | Subject-Level Analysis Dataset |
| ADAE | Adverse Event Analysis Dataset |
| ADLB | Laboratory Analysis Dataset |

## System Terms

| Term | Definition |
|------|------------|
| Asset | Any reusable artifact (code, logic, template, document) |
| Asset Card | Structured metadata record for an asset |
| Task Card | Structured record of a coding task request |
| Promotion | Moving an asset from one status level to the next |
| Draft Library | Storage for unreviewed assets |
| Validated Library | Storage for human-approved assets |
| Memory | System's accumulated knowledge from past decisions and failures |
